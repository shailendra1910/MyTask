This is a implementation of the below requirement

-- Requirements -----
Write a program that takes a basket of items and outputs its total cost.

The basket can contain one or more of the following items: Bananas, Oranges, Apples, Lemons, Peaches.

Please upload your test to Github and send me the link when completed.

-- Project Dependencies ---
Java 1.8
Junit 4.11

All dependencies are included in the maven pom file

-- Assumptions ---
The items added to the shopping basket are already of ShoppingItem type.
